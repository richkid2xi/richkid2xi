### <img src="https://api.iconify.design/material-symbols:medication-outline.svg?color=%231a6fb5" width="24" style="vertical-align: middle;"/> [Klavora](https://klavora.vercel.app/)
> Smart pharmacy inventory management for independent Ghanaian pharmacies

Klavora is a full-featured pharmacy inventory management SaaS built specifically for independent pharmacies in Ghana. Offline-first PWA with IndexedDB sync, FEFO batch deduction, Paystack Mobile Money billing, WhatsApp low-stock alerts via Termii, expired batch deletion with audit logs, and a multi-drug basket sale system. Built after real market research — visiting pharmacies, studying competitors, validating pricing.
